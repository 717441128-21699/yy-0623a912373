# 第一章 初入江湖

清晨的雾气还未散去，林远已经站在了山巅。

他望着脚下绵延的群山，心中思绪万千。三年前，他还是一个默默无闻的山村少年，每日只是砍柴种地，过着平淡无奇的生活。直到那天，一位云游的道长路过村子，看到他时忽然停下了脚步。

"此子根骨奇佳，若入我门下，日后必有大成。"

就这一句话，改变了他的命运。

他辞别了父母，跟着道长来到了这青云宗。三年苦修，如今终于到了下山历练的时候。

"林远，你在发什么呆？"身后传来一个清脆的声音。

林远回头，看到一个身着青衣的少女正快步走来。她叫苏婉，是他在宗门里为数不多的朋友。

"没什么，只是想起了一些往事。"林远笑了笑。

苏婉走到他身边，同样望着远方："下山之后，你打算去哪里？"

"先回家看看，然后到处走走。"林远顿了顿，"你呢？"

"我要去北疆，听说那里最近不太平。"苏婉的眼中闪过一丝忧虑，"据说有魔道妖人出没，已经有好几个小门派遭殃了。"

林远眉头微皱。魔道？这个词他只在宗门典籍里看到过。

"那你多加小心。"

"你也是。"苏婉从怀中取出一枚玉佩，"这个给你，若是遇到危险，捏碎它，我能感应到大概的位置。"

林远接过玉佩，只觉入手温润。他正想推辞，却见苏婉已经转身离开了。

"我们江湖再见！"

望着她离去的背影，林远握紧了手中的玉佩。他深吸一口气，转身向山下走去。

然而他不知道的是，就在他离开后不久，山巅的另一侧，一个黑衣人悄然现身，目光阴冷地注视着他的去向。

"就是他吗？"黑衣人低声自语，"宗主说的那个'特殊体质'……"

他的身影一闪，消失在了雾气之中。而林远对此一无所知，只是沿着石阶一步步向下，走向了属于他的江湖路。只是前路漫漫，等待他的，究竟是机缘还是劫难，此刻谁也说不清。
