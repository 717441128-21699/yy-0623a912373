# 第二章 山雨欲来

下山之后，林远一路向南，用了半个月的时间回到了故乡小山村。

然而眼前的景象，却让他如坠冰窟。

昔日炊烟袅袅的村庄，如今只剩下一片焦黑的残垣断壁。地上散落着焦黑的木梁和破碎的瓦罐，空气中弥漫着令人作呕的焦糊味。

"不……不可能……"

林远踉跄着冲进村子，疯狂地寻找着父母的身影。他找遍了每一间倒塌的房屋，最终在自家后院的老槐树下，找到了两具早已冰凉的尸体。

母亲的手里还紧紧攥着一件没做完的衣裳，那是她之前托人捎信说要给他做的新袍子。

"爹……娘……"

林远扑通一声跪倒在地，泪水夺眶而出。他颤抖着伸出手，却不敢去碰那两具冰冷的遗体。三年苦修，他本以为自己已经足够坚强，可在这一刻，所有的心防都土崩瓦解。

是谁干的？！

他猛然抬头，双眼赤红。村子的土墙上，留着几个用血写成的大字——"凡与青云宗有关者，皆诛之。"

青云宗……魔道……

林远的拳头握得咯咯作响，指甲深深嵌进了掌心。他想起了苏婉说过的话，魔道妖人出没北疆，可这里明明是南疆啊！

就在这时，一阵脚步声从村口传来。

林远猛地回头，只见三个身着黑袍的男子正缓步走来，脸上挂着残忍的笑意。为首那人的胸口，绣着一个血色的骷髅标记——正是魔宗的标志。

"哟，还有个漏网之鱼。"为首的黑袍人舔了舔嘴唇，"看这穿着，是青云宗的弟子吧？正好，一起上路，省得你在世上孤单。"

话音未落，三人已经同时出手，三道漆黑的掌风直扑林远面门。林远虽然悲痛欲绝，但三年苦修的本能还在。他身形一晃，险之又险地避开了攻击，同时右手探入腰间，抽出了那柄道长赠予他的青锋剑。

"你们……都得死！"

剑尖一颤，三道剑花同时绽放。这是青云宗的基础剑式"流云三叠"，林远练了不下万遍，此刻在悲愤之下使出，威力竟然比平日还要强上三分。

三个黑袍人没想到他反应这么快，为首那人闷哼一声，手臂上多了一道剑痕。

"小子，有点意思。"黑袍人眼中闪过一丝讶异，随即变得更加狰狞，"不过，就凭你一个刚入门的弟子，也想跟我们斗？"

他从怀中取出一枚黑色的令牌，正要捏碎——忽然，他的动作停住了。

因为他看到，林远的双眼，此刻已经完全变成了金色。一股前所未有的恐怖气息，正从这个少年身上缓缓升起，仿佛一尊沉睡万年的凶兽，正在苏醒。

"这……这是什么？！"黑袍人的声音，第一次带上了恐惧。
